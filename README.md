# Password Generator
A customizable wordlist, messages, and password generator.

## Installation
```gitclone https://github.com/hrtywhy/Random-Password-Generator```<br>
```cd files``` <br>
```python main.py```


## Usage
- input username
- input consonant 
- input number
- input special char
- input email destination
